# 🍗 PUBG 赛博分锅助手 (PUBG Cyber-Roasting Helper) - FrontEnd

> “打游戏不为了甩锅，那将毫无意义。” —— 某位不愿透露姓名的下等马

## 📖 项目简介

本项目是一个专为 PUBG 固定开黑小队打造的“战绩整活与赛后清算”系统的前端工程。
告别枯燥的官方战绩面板，本系统旨在通过深入挖掘游戏底层对局数据，精准识别出队伍中的“大腿”、“混子”以及“内鬼”，并自动生成适合在微信/QQ群内传阅的赛博处刑图。

**核心特色：**
* 🐎 **自动化马匹评级**：精确计算单局贡献，童叟无欺。
* 📊 **多维度战力折线图**：谁在偷偷变强？谁在持续拉胯？一目了然。
* 🏆 **离谱成就墙**：记录每一次“顺丰快递”与“人体描边”。

---

## 🛠️ 技术栈 (Tech Stack)

为了保证开发效率和顺滑的交互体验，本项目采用以下技术栈：

* **核心框架:** Vue.js (v3.x) - 稳定可靠，生态丰富。
* **构建工具:** Vite - 极速冷启动与热更新，拯救秃头。
* **路由管理:** Vue Router
* **状态管理:** Vuex / Pinia (按需选择)
* **网络请求:** Axios
* **数据可视化:** ECharts - 用于绘制高大上的“百场战力图”。
